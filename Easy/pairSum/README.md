# Find a pair with the given sum in an array

> Given an unsorted integer array, find a pair with the given sum in it.

```js
// Input:

arr = [8, 7, 2, 5, 3, 1]
sum = 10

// Output:

Pair found (8, 2)
or
Pair found (7, 3)


// Input:

arr = [5, 2, 6, 8, 1, 9]
sum = 12

// Output: Pair not found
```
